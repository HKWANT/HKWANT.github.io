css3的单位

  • em(倍数，相对于父级和自己的字号设置 例：2em;)
  • rem(倍数，相对于根级和自己的字号设置 这个要“寻根问底” 例：2rem)
  • maxwidth（设置最大宽度，超过这个宽度盒子的宽度将不会随着视窗的增大而增大）
  • maxheight(与maxwidth同义)
  • minwidth(设置最小宽度，超过这个宽度，盒子的宽度将不会随着窗口宽度的减少而减少)
  • minheight(与minheight同义)
  • vw(视窗宽度， 以视窗宽度为基准，可以用百分比表示，例：vw:50%)
  • vh(与vw同义)

hexo博客搭建

1.

- 安装nodeJS 
- 安装hexo(npm install hexo -g)并测试是否安装成功（hexo）
- 创建目录并进入（hexo init）(hexo server)
### 2. 
- 注册github账号一个
- 配置ssh-key(ssh-keygen)连续三次回车
- vim ~/.ssh/id_rsa.pub(复制ssh-key)
- github设置ssh粘贴

- 配置github与本地机器验证（git config --global user.name ""
- git config --global user.email ""）
- 新建博客域名仓库（用户名.github.io）
### 3. 
-config.yml
- deploy字段
- type:git
- repo:域名仓库的https地址
###4. 
- 发布
- npm install hexo-deployer-git - -save
- hexo g
- hexo d

done!!!
